# 🍕 Pizza Sales Analysis Project

## 📌 Project Overview

This project analyzes pizza sales data to uncover key business insights such as revenue trends, customer preferences, and best-selling products. The analysis helps in making data-driven decisions to improve sales and customer satisfaction.

---

## 🎯 Objectives

* Analyze total revenue and order trends
* Identify top-selling pizzas
* Understand customer ordering patterns
* Discover peak sales hours and days

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib & Seaborn
* Jupyter Notebook

---

## 📊 Key Insights

* 🍕 **Top-selling pizza category** contributed the highest revenue
* 📈 Sales peak during weekends and evenings
* 🧾 Most orders contain 2–3 items on average
* 💰 Certain pizza sizes generate more profit than others
